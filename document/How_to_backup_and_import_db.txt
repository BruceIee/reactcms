At first, if there is no 'data' folder under site, build it.
All backuped files are in this folder.

1. ---backup---
   under site folder, run: node aid.js -cmd=backup
   
2. ---import---
   at first you need to initdb,
   under site folder, run: node aid.js -cmd=initdb
   then import,
   under site folder, run: node aid.js -cmd=import
